# Java Basics and Object-Oriented Programming (OOP) Concepts Project

## 📌 Project Goal

The goal of this project is to understand Java programming basics and Object-Oriented Programming (OOP) concepts. This project demonstrates the implementation of loops, conditions, arrays, classes, objects, encapsulation, inheritance, and polymorphism using Java.

This project was completed as part of the Alfido Tech Internship Task.

## 🧠 OOP Concepts Used

### 1. Class and Object

A class is a blueprint for creating objects. Objects are instances of a class.

Example:  
Car is a class, and BMW is an object.

---

### 2. Encapsulation

Encapsulation is the process of hiding data and accessing it using getter and setter methods.

Benefits:

- Data security  
- Controlled access  

---

### 3. Inheritance

Inheritance allows one class to acquire properties and methods of another class.

Benefits:

- Code reuse  
- Improves maintainability  

Example:  
Employee inherits properties from Person class.

---

### 4. Polymorphism

Polymorphism allows methods to perform different tasks based on input.

Types used:

- Method Overloading (Compile-time)
- Method Overriding (Runtime)
