# Task 2 - Java Console Application

## Objective
Build a simple Java console application to practice Java fundamentals.

## Description
This program demonstrates basic Java programming concepts such as user input, conditional logic, and loops.

## Features
- Accepts user input using Scanner
- Uses conditional statements
- Uses loops for repeated operations
- Performs simple operations like managing student data or calculations

## Technologies Used
Java
Scanner
If-Else
Loops

## Learning Outcome
- Understanding Java program structure
- Handling user input
- Applying conditional logic and loops
