# Task 3 - Java Collections and Exception Handling

## Objective
Learn how to manage data using Java Collections and handle runtime errors using exception handling.

## Description
This task implements a simple Library Management System using Java Collections such as ArrayList, HashMap, and HashSet.

## Features
- Add books to library
- Display available books
- Borrow books
- Return books
- View borrowed books

## Technologies Used
Java
ArrayList
HashMap
HashSet
Custom Exception
Try-Catch

## Learning Outcome
- Understanding Java Collections
- Implementing custom exceptions
- Handling runtime errors
- Building menu-driven console applications
